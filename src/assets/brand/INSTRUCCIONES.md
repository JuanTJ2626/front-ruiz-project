# Carpeta de marca / Brand

Pon tus archivos de logo aquí con EXACTAMENTE estos nombres:

| Archivo         | Uso                                      | Formato recomendado |
|-----------------|------------------------------------------|---------------------|
| `logo.png`      | Logo completo (con texto o sin texto)    | PNG transparente    |
| `logo-icon.png` | Solo el ícono/símbolo (cuadrado 1:1)     | PNG transparente    |
| `favicon.png`   | Favicon del navegador (mínimo 32×32px)  | PNG transparente    |

## Notas
- Si solo tienes un archivo, ponlo como `logo.png` y el código usará ese mismo en todos lados.
- El favicon también necesita copiarse a /public/favicon.svg (o cambia la extensión en index.html).
- Tamaños recomendados: logo.png → 400×120px, logo-icon.png → 128×128px
